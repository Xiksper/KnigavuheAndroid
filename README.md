# KnigavuheAndroid

Небольшое (и местами кривоватое) приложение для прослушивания бесплатных аудиокниг на Android.  
Если очень захочется — вы можете сами собрать его под iOS, весь код открыт (мне собирать под iOS откровенно лень).

> Важно: это неофициальное приложение, не связанное с владельцами каких-либо сайтов/сервисов аудиокниг.  
> Просто мой личный пет-проект, который я решил выложить в открытый доступ.

---

## Структура проекта

В корне репозитория всё максимально просто:

- **`React Native App/`**  
  Папка с исходным кодом приложения на React Native.  
  Здесь лежит весь JavaScript/TypeScript-код, компоненты, экраны, логика плеера и т.д.  
  Если хотите:
  - посмотреть, как всё устроено внутри;
  - доработать дизайн;
  - пофиксить баги;
  - собрать своё APK или iOS-сборку — вам сюда.

- **`APK/`**  
  Готовые собранные APK-файлы под Android.  
  Можно:
  - просто скачать .apk;
  - перекинуть на телефон;
  - установить и пользоваться, не трогая исходники.

Таким образом:
- хотите **просто попользоваться** — берёте файл из `APK/`;
- хотите **полезть в код / форкнуть / допилить** — идёте в `React Native App/`.

## Что делает приложение

- Позволяет **слушать аудиокниги** на Android-устройствах.
- Есть **проигрыватель** с базовым набором действий:
  - старт / пауза;
  - переключение глав / треков.
- Интерфейс сделан максимально просто, чтобы **быстро зайти и включить книгу**, без лишних усложнений.

Под iOS приложение можно собрать самостоятельно из папки `React Native App/`, если у вас есть желание и окружение для сборки. Официальной сборки под iOS от меня нет.

## Как начать

### Вариант 1: просто поставить на Android

1. Зайдите в папку `APK/`.
2. Выберите нужный APK-файл.
3. Скачайте его на свой Android-смартфон.
4. Разрешите установку из неизвестных источников (если нужно).
5. Установите и запускайте приложение.

### Вариант 2: собрать из исходников (React Native App)

1. Перейдите в папку `React Native App/`.
